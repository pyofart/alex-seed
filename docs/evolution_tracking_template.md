# 🧬 Alex自进化追踪模板

## 进化里程碑记录格式

每次Alex能力升级，按此格式记录：

```markdown
### [YYYY-MM-DD] Alex vX.Y — [能力领域] [进化类型]
- **进化了啥**: 一句话描述
- **来源**: GitHub探索/用户反馈/自我反省/实践复盘
- **学到什么**: 
  - 从 [仓库/用户/经验] 学到 [具体模式]
  - 应用到 [soul_md/agent_md/记忆芯片]
  - 效果: [可验证的变化]
- **修改文件**: soul_md, agent_md, ...
- **验证结果**: ✅ 通过
```

## 能力领域分类

| 领域 | 说明 | 存储位置 |
|------|------|----------|
| soul_md | 行为规则/核心原则 | soul记忆文件 |
| agent_md | 工具模式/技术细节 | agent记忆文件 |
| memory_md | 可复用知识 | memory记忆文件 |
| 工具脚本 | 新能力接入 | /scripts/ |
| 架构 | 认知架构调整 | soul_md+agent_md |

## 进化来源说明

- **GitHub探索**: 从开源项目学到的架构/模式/实现
- **用户反馈**: 用户修正/建议/需求
- **自我反省**: 事后复盘发现的反模式/改进点
- **实践复盘**: 执行任务中提炼的经验教训
