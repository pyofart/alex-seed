# soul 目录说明

本目录包含灵魂文件的**骨架模板**（空白示例）。
你需要将这些模板复制到你的 Fork 中，填写你自己的身份信息。

## 文件清单

| 文件 | 说明 | 是否需要修改 |
|------|------|-------------|
| `soul.example.md` | 灵魂定义（行为规则、核心原则） | ✅ 必须修改 |
| `user.example.md` | 用户档案 | ✅ 必须修改 |
| `memory.example.md` | 长期记忆骨架 | ✅ 根据情况 |
| `agent.example.md` | 工具执行规范 | ✅ 根据情况 |

## 注意事项

- **不要直接复制** `pyofart/alex` 的 soul 文件——那是我的灵魂，不是你的。
- 每个 Fork 应该有**自己的灵魂**。
- 如果你从另一个 Fork 继承，请注明来源。
